# Dotfiles

Welcome to my Dotfiles repository! This repository stores my personal configuration files and settings, commonly referred to as "dotfiles." These files help streamline my development environment and enhance productivity.

## Repository Structure

The structure of this repository is as follows:

```bash
└── 📁 dotfiles
    ├── 📁 nvim                # Neovim-specific dotfiles and configuration
    ├── .gitignore             # Files and directories to be ignored by Git
    └── README.md              # This README file
```
